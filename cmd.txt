━════╗• Команды •╔════━

.к/(твой префикс) инфо - Покажет профиль в Sakura.
.к/(твой префикс) инфо подробно - Показывает подробную информацию.
.к/(твой префикс) стата - Покажет инфу о странице.
.к/(твой префикс) ник [ник] - Устанавливает ник в боте.
.к/(твой префикс) агенты - Список всех агентов.
.к/(твой префикс) +ава - Устанавливает аватарку.
.к/(твой префикс) -ава - Удаляет аватарку.
.к/(твой префикс) тема - Меняет тему чата.
.к/(твой префикс) лс - Отправляет смс пользователю в лс
.к/(твой префикс) +/-чс - Кидает в чс пользователя/Убирает с чс пользователя.
.к/(твой префикс) добавить - Добавляет пользователя в чат.
.к/(твой префикс) +/-админ - Выдает права администратора в чате/Забирает права администратора в чате.
.к/(твой префикс) кик - Исключает пользователя из чата.
.к/(твой префикс) дд [число] - Удаляет ваши сообщения из чата.
.к/(твой префикс) +/-дов - Добавляет пользователя в доверенные/Удаляет пользователя с доверенных.
.к/(твой префикс) довы - Показывает список доверенных.
.к/(твой префикс) +/-др - Отправляет заявку в друзья/Отменяет заявку в друзья.
.к/(твой префикс) !др - Удаляет друзей по 100 человек.
.к/(твой префикс) !собак - Чистит собачек с друзей.
.к/(твой префикс) !исток - Чистит исходящие уведы
.к/(твой префикс) +/-игнор - Удаляет для вас все смс пользователя/Перестанет удалять смс игнорируемого.
.к/(твой префикс) игнор - Показывает список игнорируемых.
.к/(твой префикс) +/-изб - Добавляет пользователя в избранные/Удаляет пользователя с избранных.
.к/(твой префикс) изб - Показывает список избранных.
.к/(твой префикс) +/- лайк - Лайкает аву/ Убирает лайк с авы.
.к/(твой префикс) пролайкать - Лайкает 5 постов пользователя.
.к/(твой префикс) пинг - Показывает скорость ответа бота.
.к/(твой префикс) !рег - Регистрирует нового пользователя.
.к/(твой префикс) шаги - Накрутка шагов.
.к/(твой префикс) линии [ прекрепляя фото ] - Делает линии на фотогафии.
.к/(твой префикс) статус - Устанавливает статус на страницу.
.к/(твой префикс) стикеры - Покажет платные стикеры пользователя.
.к/(твой префикс) слова - Покажет текст трека.
.к/(твой префикс) +/-шаб - [название] - Создает шаблон/[название] - Удаляет шаблон.
.к/(твой префикс) шабы все - Отображает количество шаблонов.
.к/(твой префикс) +/-гп - [ссылка] - Добавит в группу/[ссылка] - Выйдет из группы.
.к/(твой префикс) !группы - Чистит группы.
.к/(твой префикс) админы - Показывает список админов.
.к/(твой префикс) владелец - Показывает список владельцев.
